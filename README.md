Project: High-Performance Local DB Environment

Overview:
This project focuses on building a containerized PostgreSQL environment optimized for Apple Silicon (M3 Max). The goal is to explore database performance and storage logic in a virtualized environment.

Stack:

Engine: PostgreSQL 15+

Orchestration: Docker & Docker Compose

Logic: Python (psycopg2)

Hardware: MacBook Pro M3 Max (Testing local I/O performance)

Why this matters:
I'm interested in how containerization affects low-level system resources and disk I/O, especially on ARM-based architectures.
